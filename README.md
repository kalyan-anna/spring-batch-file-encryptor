# spring-batch-file-encryptor

## Technologies
* Spring Boot 
* Spring Batch

## Install
mvn install

## Run
Command line arguments
* numberOfThreads (1 - 20)
* pathToInputFile

mvn spring-boot:run -Dspring-boot.run.arguments=--numberOfThreads={noThread},--pathToInputFile={path}
